## BRICS Unit Basket Methodology

### Basket Composition
The BRICS Unit Basket v1 is composed of:

| Component | Weight | Genesis Rate (per USD) |
|-----------|--------|------------------------|
| **Gold** | 40% | 0.9823 grams per unit |
| **BRL** (Brazilian Real) | 12% | 4.92 |
| **RUB** (Russian Ruble) | 12% | 89.50 |
| **INR** (Indian Rupee) | 12% | 83.20 |
| **CNY** (Chinese Yuan) | 12% | 7.18 |
| **ZAR** (South African Rand) | 12% | 18.40 |

### Calculation Methodology

1. **Gold Component (40% base weight)**
   - Unit base: 0.9823 grams of gold per unit
   - Gold price: TWAP from spot market (USD per troy ounce)
   - Conversion: 31.1034768 grams per troy ounce

2. **Currency Component (60% total - 12% each)**
   - Each currency contributes based on: `FX_WEIGHT * (current_rate / genesis_rate)`
   - Genesis rates frozen at: 2026-01-25T00:00:00Z
   - TWAP window: 5 seconds (5000ms)

3. **Unit Price Formula:**
   ```
   basketIndex = GOLD_WEIGHT + Î£(FX_WEIGHT * (current_FX / genesis_FX))
   unitGoldGrams = UNIT_BASE_GOLD_G * basketIndex
   unitUSD = unitGoldGrams * (goldTWAP / GOLD_G_PER_OZ)
   ```

### Key Technical Details

- **TWAP Window:** 5 seconds (rolling window)
- **Methodology Frozen:** 2026-01-25T00:00:00Z (UTC)
- **Hash Verification:** SHA-256 hashes of genesis.json and chart library
- **Server Port:** 3000 (default)
- **Endpoints:**
  - `/latest.json` - Current unit price data
  - `/meta.json` - Metadata and hashes
  - `/genesis.json` - Genesis configuration
